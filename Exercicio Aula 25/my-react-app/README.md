# Minha Aplicação React

Esta é uma aplicação React simples criada com Vite, utilizando Bootstrap para estilização.

## Componentes

- **Navbar**: Barra de navegação com links para Home, Sobre e Contato.
- **Main**: Área principal com título e parágrafo.
- **Footer**: Rodapé com um parágrafo simples.
